\beginoef
Stel dat je een karakterteken wil omzetten naar een string (van lengte 1). Argumenteer waarom welk stukje code (niet) werkt.
\begin{verbatim}
	char c = 'x';
	string s = "" + c;
	cout << "karakter "<< c <<" omgezet: "<< s << "." << endl;

	char k = 'y';
	string w = "";
	w += k;
	cout << "karakter "<< k <<" omgezet: "<< w << "." << endl;
\end{verbatim}
